Hiking App 2021

The overall description

This project was for my CSE 248 class. The goal of this project was to construct a functional hiking app (although real data was not a requirement). The app itself would act similar to many out there.

Basic requirements

It had to have admins and regular users, in which admins had more power over the website than the users. the users(including admins) could search for trails via any part of the trail. the users could also have a hiking history that kept track of the trails they've hiked.
